classifier

This model was trained in full autonomy by a team of Claude Code agents. It's a small piece of the swarm, intended to classify input message as one of 4 classes: Time, Weather, Other, or Swarm. The tag is used by the Hive to inject relevant data into the drone's prompt.
Python for training — HuggingFace Transformers, PyTorch Rust for inference — Candle compiled to WASM via wasm-pack
